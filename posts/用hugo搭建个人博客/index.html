如何使用 hugo 搭建个人博客

How to make personal blog via hugo （for Win users）

  1. 用 hugo 搭建博客的第一步，就是学习使用 Markdown 语法，这个语法非常简单，很快就能上手。
  • Before you create your blog，you need to learn how to use markdown. It‘s very easy.

安装 hugo

install hugo

  1. https://github.com/gohugoio/hugo/releases ，找到 hugo_XXX_Windows 64bit.zip，我选择了大约 40M 的那个。下载并放到 D 盘的 Software 路径下，具体格式为 D:\Software\hugo\hugo.exe ;若没有此路径可以自行创建。
  • you need to open the link and find a document names hugo_XXX_Windows 64bit.zip and download. You can create the directory as “D:\Software\hugo\hugo.exe”.
  1. 将创建好的路径添加到 path 中：右键点击此电脑——点击属性——点击高级系统设置——点击环境变量——在下方的系统变量中找到 path——编辑——新建——输入复制好的路径。
  • Add the path :click the right mouse button on ‘this computer (a computer shaped icons) - Properties - Advanced System Settings - environment variables - find the world ‘path’ in the system variables - Edit - create - enter the path you created just now.
  1. 重启，在 git bash 或其他可以输入命令行的地方运行下列命令，若可以看到版本号，则安装成功。这里可能遇到一个问题，vscode 新建的终端需要是 bash 状态，如果是 powershell 的话，可以百度解决。
  • Reset your computer , and run the command . If the version number appears ,that means you did it successfully .PS: You need to be in Bash state in vscode ,rather than Powershell.

hugo version

搭建 blog

Set up your blog

  1. https://gohugo.io/ 进入 hugo 官网，点击 Quick Start。第一步 install（安装）已经完成，所以跳过。
  • Enter in the official website of hugo and click Quick Start. We’ve already installed the software，so please ignore the first procedure.
  1. 创建新的站点：首先需要新建一个目录，然后在当前目录打开 git bash 或者 cmder，输入下列命令并回车。
  • Create a new site : create a new directory and open git bash or cmder in current directory , input the command below (你的 GitHub 用户名 means your GitHub username).

hugo new site 你的GitHub用户名.github.io-creator

  1. 在 vscode 中打开这个目录
  • Open the directory below in vscode. (你的 GitHub 用户名 means your GitHub username).

code hugo new site 你的GitHub用户名.github.io-creator\

  1. 接下来的三个步骤都比较简单，按照教程输入即可
  • Do as the Quick Start.

git init

git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

echo 'theme = "ananke"' >> config.toml

  1. 接下来就是创建博客文档了。创造完毕之后可以在 vscode 的 content 文件夹下看到自己新建的博客。
  • The next step is to create a blog document. after creation, you can see your new blog in the content folder of vscode .(你想要的名字 means the name that you want).

hugo new posts/你想要的名字.md

写博客

Write your blog

  1. 按照 Markdown 语法分别设置各级标题以及正文等内容，并将 draft 的状态改为 false，因为它并不是一个草稿了，你需要将它作为正式文档上传。
  • Set the title and text of each level according to markdown syntax, change the status of draft to false.
  1. 创作完成后，在 vscode 的终端运行：
  • Run the command below after finishing your blog.

hugo server -D

  1. 运行完毕后，可以在终端控制台上看到一个连接，使用 Ctrl+单击进入，千万别按 Ctrl+C，否则会终止。
  • When you see a link in console ,click it.

预览博客并修改基本配置

Preview and changes

  1. 完成第三部的第三点后，你就可以在网页上预览自己的博客了，不仅可以检查电脑版，还可以右键检查是否适配与手机。
  • Inspect your blog.
  1. 回到 vscode，找到 config.toml 文件，将第二行的 language 改为"zh-Hans”，即简体中文。
  • Skip it .
  1. 将第三行的标题改为自己喜欢的
  • Change the title according to your preference.
