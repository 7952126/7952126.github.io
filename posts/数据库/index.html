sql 语言

  1. 查所有用户：select * from user
  2. 数用户：select count（*） from user
  3. 看地址 ：select * from user where ADDRESS =shanghai
  4. 看多少用户在上海：select count（*）from user where ADDRESS =shanghai
  5. 用* 这种方式查询所有的列
  6. 查具体的列：select id, name from user where ADDRESS =shanghai
  7. Order by (以及按降序排序) select id, name from user where ADDRESS =shanghai order by id desc
  8. 分页 select _ from user limit (offset 从第几个元素开始查找,limit 最多返回几个元素)  Eg：select _ from user limit 2,3
  9. 看看每个地区有多少用户：group by  select ADDRESS from user group by ADDRESS
  10. 每个城市有多少人 ：select ADDRESS ,count(*) from user group by ADDRESS
  11. Order 是 sql 关键字，所有要加反引号。获取订单表：select * from ORDER
  12. 每个商品被下了多少单 select GOODS_ID ,count(*) from ORDER group by GOODS_ID
  13. 改列名 select GOODS_ID ,count(*) as count from ORDER group by GOODS_ID
  14. 把商品按照 id 分组，并求总的成交金额 Select GOODS_ID,sum（ GOODS_NUM*GOODS_PRICE ） as TOTAL from ORDER group by GOODS_ID order by TOTAL desc
